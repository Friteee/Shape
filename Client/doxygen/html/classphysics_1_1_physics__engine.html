Detailed Description

Class used for collision detection and physical objects updates.

One uses add_static_polygon to add static objects for collision detection, add_moving_polygon to add moving objects for collision detection and detected collision handling, delete_static_polygon to delete a static object delete_moving_polygon to delete a moving object run to handle collisions and run physics calculations reset to delete every physics object (both static and moving) from collision detection
